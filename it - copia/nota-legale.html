Cookie policy

www.topciment.com è un sito web gestito da GRUPO NEGOCIOS PO, SLU

I cookie sono file inviati a un browser da un server web per registrare le attività degli utenti sul sito web. I cookie utilizzati da www.topciment.com o da terzi che agiscono per suo conto sono associati solo ad un utente anonimo e al suo computer e non forniscono i dati personali dell'utente. Tutte queste informazioni vengono registrate in un'attività di file server che consente la successiva elaborazione dei dati per ottenere misure statistiche che consentano di conoscere il numero di impressioni delle pagine, il numero di visite ai servizi web, ecc.

Attualmente stanno raccogliendo i dati utilizzando i seguenti strumenti: Google Analytics.

I cookie utilizzati in questo sito web non raccolgono in alcun momento i dati personali (nomi, cognomi, indirizzo o dati bancari) con gli strumenti e non lo faranno mai.

Utilizzando questo sito, accetti i criteri dei cookie.

 


Informazioni su GOOGLE ANALYTICS

Questo sito web utilizza Google Analytics, un servizio di analisi web fornito da Google Inc., società di Delaware con sede a .1600 Amphitheatre Parkway, Mountain View (California), CA 94043, United States ("Google").

Google ha sviluppato un plug-in per disattivare l'invio di dati a Google Analytics nei browser e può essere scaricato qui: https://tools.google.com/dlpage/gaoptout?hl=es


Dettaglio dei COOKIES usati:

Fonte: https://developers.google.com/analytics/devguides/collection/gajs/cookie-usage?hl=es-ES

Cookie Name Default Expiration Time Description
_ga 2 years Used to distinguish users.
__utma 2 years from set/update Used to distinguish users and sessions. The cookie is created when the javascript library executes and no existing __utma cookies exists. The cookie is updated every time data is sent to Google Analytics.
__utmb 30 mins from set/update Used to determine new sessions/visits. The cookie is created when the javascript library executes and no existing __utmb cookies exists. The cookie is updated every time data is sent to Google Analytics.
__utmc End of browser session Not used in ga.js. Set for interoperability with urchin.js. Historically, this cookie operated in conjunction with the __utmb cookie to determine whether the user was in a new session/visit.
__utmz 6 months from set/update Stores the traffic source or campaign that explains how the user reached your site. The cookie is created when the javascript library executes and is updated every time data is sent to Google Analytics. 

Spiegazione: 

Fonte: https://trucosgoogleanalytics.com/funcionamiento-cookies-google-analytics-614/#axzz2JMZ52kHs

_ga / usato per differenziare gli utenti. (Richiede 2 anni per scadere).

__utma / questo cookie genera un utente unico, utilizzato per contare quante volte visitano il sito un ID utente. Ricorda anche quando è stata la prima e l'ultima volta che indossava il web.
Questo cookie è persistente e può essere considerato che (scadono due anni) non scadrà mai.

__utmb - __utmc / 2 cookies vengono utilizzati per calcolare quando una sessione è terminata. Il B registra l'orario di arrivo alla pagina mentre il C verifica se la sessione aperta dovrebbe essere o devi creare una nuova sessione.
L'utmb scade dopo 30 minuti dall'ultimo record nella vista pagina mentre l'utmc è una variabile di sessione, quindi viene automaticamente cancellata quando si cambia il Web o quando si chiude il browser.
Quando la pagina viene caricata se l'utmc non esiste, si considera una nuova sessione. Se esiste ancora, la sessione aperta viene mantenuta.
È possibile calcolare solo l'orario sulla pagina fino alla penultima visualizzazione pagina, poiché viene eseguita la sottrazione tra il valore di B nella pagina corrente e il valore della B sulla pagina precedente per calcolare l'ora.

__utmz / il quarto cookie è responsabile della registrazione dell'origine dell'utente, così come delle parole chiave. Ha una durata di 6 mesi, ma viene rinnovata ogni volta che visita il sito web a condizione che non sia una visita diretta.

 

Come faccio a controllare quali cookie sono installati sul mio dispositivo?

È possibile configurare il browser per essere notificato sullo schermo della ricezione dei cookie e accettarli o meno, nonché per impedire l'installazione sul disco rigido per impostazione predefinita. Questo sito web funziona correttamente anche se decidi di non accettare nessuno dei nostri cookie.

Dipende se accettate i cookie o no. Un modo per farlo è attraverso le impostazioni del browser Web. La maggior parte dei browser Internet ti permette di controllare quasi tutti i cookie attraverso le impostazioni del tuo browser. (Se si utilizzano le impostazioni del browser per bloccare tutti i cookie, è possibile che non è possibile accedere a determinate sezioni del nostro sito.) I seguenti siti web offrono informazioni per configurare i cookie in alcuni browser più conosciuti:

  • Apple Safari
  • Google Chrome
  • Microsoft Internet Explorer
  • Mozilla Firefox
