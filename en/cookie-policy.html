_ga / used to differentiate users. (takes 2 years to expire).

__utma / this cookie generates a unique user, which is used to count how many times they visit the site a user ID. It also records when he was the first and the last time she wore the web.
This cookie is persistent and can be considered that (it takes 2 years expire) will 'never' expire.

__utmb - __utmc / 2 cookies are used to calculate when a session is finished. The B records the time of arrival at the page while the C check if the open session should be or you must create a new session. 
The utmb expires after 30 minutes from the last record in page view while the utmc is a session variable, so it is automatically deleted when you change Web or when you close the browser. 
When the page is loaded if the utmc does not exist, it is deemed a new session. If it still exists, the open session is maintained. 
You can only calculate the time on page until the penultimate page view, since subtraction between the value of B in the current page and the value of the B on the previous page is done to calculate the time.
