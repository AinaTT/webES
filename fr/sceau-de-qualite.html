Sceau de Qualité

Ce Sceau de Qualité peut être affiché seulement sur les sites Web des entreprises qui ont décidé de s'engager à travailler avec Topciment, en respectant les normes de qualité les plus élevées.


  • 1.- Sélectionnez le code et appuyez sur le bouton de copie (ou CTRL + C)

  • 2.- Collez le code HTML où le sceau apparaîtra sur votre site web (de préférence sur la page d'accueil en haut). Note: Ne changez pas ou ne modifiez pas le code HTML car cela pourrait empêcher le bon fonctionnement de votre lien. Il est très important aussi de respecter la proportion des images.

  • Une fois que vous avez inséré le code de notre sceau et que vous avez vérifié que le lien mène à www.topciment.com, dites-nous.
